# WinFXNet lic patcher
WinFXNet license patcher

Я нашёл время на праздниках (всех с прошедшим Днём народного единства), сел за отладчик и выявил недостающие пазлы.
На свёт получились три программы.

Программа для извлечения строки, полученной путем преобразования содержимого файла WinFXNet.lic

## License Viewer. 
Простая программа, которая выведет вам строку, полученную расшифровкой по алгоритму. Тут уже обновленная версия, так как непечатаемые символы тоже играют роль.

![License Viewer](/Images/Lic-Viewer.jpg "License Viewer")

## Date Patcher.
У вас уже есть просроченная лицензия и вам нужно просто продлить до необходимой даты (в данном релизе я захардкодил дату 2050-01-21). Скачиваете, в папку с программой помещаете ваш файл WinFXNet.lic, запускаете программу - на выходе получаете файл WinFXNet_patched.lic.

![Date Patcher](/Images/Date-Patcher.jpg "Date Patcher")

## All Patcher.
У вас уже есть просроченная лицензия (если нет, то воспользуйтесь моим файлом, хотя думаю его вам за глаза хватит) и вам захотелось поменять имя компании, имя пользователя, номер лицензии и её дату? Тогда в папку с программой помещаете ваш файл WinFXNet.lic, запускаете программу, вводите все нужные вам данные и получаете ваш именной файл. Можно запустить с параметрами, если вам лень вбивать все строки руками.


Скомпилированные варианты первых двух программ лежат в релизах - смотрите самый последний.
